# SchoolProfile

SchoolProfile is a simple web application for managing and displaying school profiles. It includes features such as adding, editing, and viewing school information.

## Features

- Add and manage school profiles
- Edit existing school information
- View detailed school profiles

## Installation

To get a local copy of this project up and running, follow these steps:

### Prerequisites

- Ensure you have [Git](https://git-scm.com/) and [Node.js](https://nodejs.org/) installed on your machine.

### Clone the repository

```bash
git clone https://github.com/yusufdimari/SchoolProfile.git
cd SchoolProfile
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm start
```

### Open the application

Visit `http://localhost:3000` in your web browser to see the app in action.

## Usage

Once the application is running, you can:

1. Add a new school profile by navigating to the "Add School" section.
2. Edit an existing school profile by selecting the school and clicking "Edit."
3. View detailed information about a school by selecting it from the list.

## Technologies

This project is built with the following technologies:

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
